Enable users to upload and customize their avatars directly from your WordPress website. Replace the default gravatar dependency with a user-friendly avatar uploader.

== Description ==

MBT Avatar Uploader is a plugin designed to overcome the limitations of WordPress's default gravatar-based avatar system. With this plugin, users can upload custom profile pictures or avatars directly from their devices, customize their size and style, and enhance the personalization of their profiles.

**Features:**
- Upload avatars from the frontend or user profile page.
- Supports JPG, JPEG, and PNG file formats.
- Crop and style avatars with circle or square options.
- Set maximum file size and dimensions for avatar uploads.
- Use shortcodes `[MBT_Avatar_Field]` and `[MBT_Avatar]` to display uploader and avatars anywhere.

== Installation ==

1. Download the plugin zip file.
2. Go to your WordPress site and navigate to `Plugins > Add New`.
3. Upload the downloaded zip file and click "Install Now."
4. Activate the plugin.

== Usage ==

**Shortcodes:**
- `[MBT_Avatar_Field]`: Display the avatar uploader form with an "Upload Avatar" button.
- `[MBT_Avatar]`: Display the user avatar wherever this shortcode is used.

**Plugin Settings:**
Navigate to `Users > Avatar Uploader` in your WordPress admin to configure:
1. **Max Avatar Size Allowed**: Restrict avatar uploads by file size.
2. **Allowed File Type**: Restrict uploads to specific file formats (JPG, JPEG, PNG).
3. **Avatar Style**: Choose between circle or square display styles.
4. **Avatar Dimension**: Define the dimensions for avatars.

== Frequently Asked Questions ==

= What file types are supported? =
The plugin supports JPG, JPEG, and PNG formats.

= How can I display the uploader on a page? =
Use the `[MBT_Avatar_Field]` shortcode to display the uploader.

= Can I customize the avatar size and style? =
Yes, you can define the maximum size, dimension, and style (circle or square) in the plugin settings.
